AboutSM-2 spaced repetition
Anki-style spaced repetition using the classic SM-2 algorithm (ease factor 2.5 → 1.3, learning steps 1m / 10m, graduating to 1 day). Read the romaji prompt, sketch the kana in the practice box, then reveal and grade yourself honestly.
  • Space — show answer
  • 1/2/3/4 — Again / Hard / Good / Easy
  • Ctrl+Z — undo last chalk stroke
Card formatting (face & answer fields):
  • \ — line break (separator is dropped); the only auto-wrap character
  • (text) at the end — small reading under the main text
    e.g. 猫 (ねこ) shows 猫 with ねこ beneath it
  • Combine freely: 猫\犬 (ねこ\いぬ) → two main lines, two reading lines
